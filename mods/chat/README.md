# Chat app
Similar to Telegram but federated

## Features Roadmap
### Stage 0
- Standard Chats
    - Create DMs, groups
    - Send+Receive messages
    - Attachments

### Stage 1
- Standard chats:
  - Reply to message
  - Edit message
  - Delete message
    
- Secret chats:
  - Full E2E
  - Not stored on server

### Stage 2
- Idea: Public chats
    - Effectively a forum
- Chat folders
  - invite link that invites user to all chats in the folder
