# Built-in Apps
These apps come pre-installed. They integrate with each-other and other apps.

## Important Built-in apps
- Contacts
  - Profile pages for users
- Notifications
  - From apps for users
- Chat
  - Federated chat app
- Storage
  - Internal file storage for apps

## Non-federated mods
Fediy wrapper for basic server functionalities. Benefits could include requiring local authentication to access content.

No modification to webapps would be required but could be enhanced with user data for example.

This would also make fediy play nicely with other services on the same domain. (ie - homepage, blog.example.com, etc.).

### Examples
- Static server
- Reverse Proxy
- Redirect
