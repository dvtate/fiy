# Contacts app
Manage contacts for federated users. 

## Features Planning
### Stage 1
- Add contact
- Edit contact
- Edit personal contact
- Delete contact
- List contacts
- Search contacts
- Web interface + API

### Stage 2
- User public profiles
  - Let them determine what info to share publicly
  - Let them share their full contact info with other users
  - Field components
- Send your contact info to other users
- Accept/merge received contacts
- Download vCard file for contact

### Stage 3
- CardDAV: https://datatracker.ietf.org/doc/rfc6352/

### Stage 4
- i18n
